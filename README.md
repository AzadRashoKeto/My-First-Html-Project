# My First HTML Project

This is a simple beginner project built with **HTML**, **CSS**, and **JavaScript**.  
It was created using **Visual Studio Code** and managed with **Git** and **GitHub**.

## Project Overview
The project displays a basic webpage with:
- A heading and a short description.
- A button that changes the background color randomly when clicked.

## Files
- `index.html` → The main HTML structure.
- `style.css` → Styling for the page (colors, fonts, layout).
- `script.js` → JavaScript functionality (background color change).
